В папках есть набор есть набор vtk файлов. Это файлы - результат виртуальных экспериментов и их можно открыть в программе Paraview,
чтобы увидеть их визуальное представление и анимацию. Частая задача результатов экспериментов их последующая обработка.

В данном случае в файлах - координаты частиц диаметром 1 с указанием векторов направления.
С точки зрения смысла это модель магнитных частиц с магнитным моментом.
В результате экспериментов они сгруппировались в клубки. Ваша задачи выделить их.

Кластеризация типичная задача машинного обучения, но здесь все проще, потому что можно ввести детерминированный критерий кластера.
Частицы образуют кластер если квадрат расстояния между частицами с номерами i и j меньших 1.2 или свойство
cluster_num у двух частиц совпадают. Четкий критерий позволяют использовать теорию графов
и выделение компонентов связности для решения задачи кластеризации.

Критерий в терминах кода: pp_dist[i,j] <= dth or cluster_num[i] == cluster_num[j]

В папке есть clusters_for_chains_from_vtk.py его нужно дописать,
так чтобы в результате записывались файлы: отдельный файл для каждого кластера с координатами частиц.

Затем использовать threading, multiprocceing, mpi для параллельной обработки файлов vtk и измерения времени работы.
